# Recipe_Recommendation_System

<h2>Project Description</h2>

The Recipe Recommendation System is a personalized application that helps users discover recipes based on their dietary preferences, health conditions, allergies, and nutritional goals. Users can create an account, provide details such as weight, health concerns, and allergies, and then log in to receive filtered recipe suggestions. The system allows users to search for recipes using filters like protein level, carbohydrate level, preparation time, difficulty, and ingredients. It ensures that recipes containing ingredients related to the user’s allergies are excluded. Users can view recipe details, maintain a history of selected recipes, and provide reviews and ratings for recipes they have tried. The project is developed using Java Swing for the user interface, MySQL for the database, and JDBC for database connectivity.

<h2>Database Components</br></h2>
Main tables used in the project include:</br>
User – stores user information</br>
recipes – stores recipe details</br>
ingredients – stores recipe ingredients</br>
selected – stores user recipe history</br>
reviews – stores ratings and feedback</br>
user_health_concerns – stores user health conditions</br>
user_allergies – stores user allergies</br>
